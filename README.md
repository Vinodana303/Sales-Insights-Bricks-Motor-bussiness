# Sales-Insights-Dashboard
<h3>📌Project Overview</h3>

This project presents an interactive Sales Data Analysis Dashboard built using SQL, Power BI, and Excel.
It provides insights into revenue, sales quantity, customer performance, and product trends across multiple markets.

The dashboard helps stakeholders make data-driven decisions by identifying key revenue drivers and performance patterns.  

<h2>🚀 Key Features</h2>

- 💰 Total Revenue Analysis – 984.81M overall revenue

- 📦 Sales Quantity Tracking – 2M total sales

  - 🌍 Market-wise Performance

  -  Revenue by markets

  -  Sales quantity by markets

- 📈 Revenue Trends

  - Time-series analysis across years (2017–2020)

- 👥 Top Customers

  - Top 5 customers contributing to revenue

- 🛍️ Top Products

  - Best-performing products based on revenue
